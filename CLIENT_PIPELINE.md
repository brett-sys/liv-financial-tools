# New Client Pipeline — Cheat Sheet

> One page. Print it, pin it, hand it to every agent. Every client lives in **exactly one
> stage** at a time. The job is simple: do the action, move the stage.

---

## Paste this into GHL

In **Go High Level → Opportunities → Pipelines → Create/Edit Pipeline**, add these stages in
this order:

```
New Lead
Attempted
Contacted
Appointment Set
Presented / Quoted
Application Submitted
Issued / Active
Referral Requested
```

Two "off-ramp" stages keep the main board clean (clients that aren't moving forward):

```
Not Interested
Lost / Lapsed
```

---

## What each stage means & what moves it forward

| Stage | A client is here when… | Action that moves them to the next stage |
|-------|------------------------|------------------------------------------|
| **New Lead** | Lead just arrived, no dial yet | Dial within `[X minutes]` → **Attempted** |
| **Attempted** | You dialed, no live contact | Keep the call/text cadence until you connect → **Contacted** |
| **Contacted** | You had a live conversation | Book a time → **Appointment Set** |
| **Appointment Set** | It's on the calendar | Run the appointment + show a quote → **Presented / Quoted** |
| **Presented / Quoted** | They've seen an illustration/quote | Get the yes + start the app → **Application Submitted** |
| **Application Submitted** | E-app sent to carrier | Carrier approves & policy in force → **Issued / Active** |
| **Issued / Active** | Policy is active | Deliver policy + ask for referrals → **Referral Requested** |
| **Referral Requested** | You've asked for referrals | Log referrals in the Referral Tracker (cycle restarts for each new lead) |

**Off-ramps (use anytime):**

| Stage | Use when… |
|-------|-----------|
| **Not Interested** | They said no / not a fit right now (revisit later) |
| **Lost / Lapsed** | Policy didn't issue, was declined, or lapsed |

---

## The 30-second rules

- **One stage per client.** If you're not sure where it goes, it goes to the *earlier* stage.
- **Move the card the moment the action happens** — not at end of day.
- **Speed-to-lead:** first dial within `[X minutes]`. This is the single biggest needle-mover.
- **Log every attempt** in the Call Logger so the board reflects reality.
- **A stalled card is a flag**, not a failure — that's where your upline can actually help.

---

## Tools at each stage

| Stage | Tool |
|-------|------|
| New Lead → Contacted | **Lead Manager** (claim/import) + **Call Logger** (Agent Toolkit) |
| Presented / Quoted | **Underwriting tool** → **Quoter** → **PDF Generator** (illustration auto-uploads to GHL) |
| Application Submitted | E-app / carrier portal + "Policy Submitted" PDF |
| Referral Requested | **Referral Tracker** (Agent Toolkit) |
| All day, every stage | **Scoreboard** (activity + outcomes) |

*If it's not in the pipeline, it isn't a deal. Move the card.*
